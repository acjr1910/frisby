# @frisby/containers

Functional Programming Containers/Monads.

## Installing

@frisby/containers can be installed through npm or yarn:

```
$ npm install @frisby/containers
```

```
$ yarn add @frisby/containers
```

## Usage

Import the functions that you want to use from @frisby/containers.
@frisby/containers supports tree-shake.

```javascript
import { compose, map } from @frisby/containers
```
